Overview

BUYCartLineItem is a subclass of BUYLineItem that extends the object by exposing the BUYProductVariant that the line item was initialized with using initWithVariant:.

Note that this object is only used for a BUYCart and line item objects on BUYCheckout are represented by BUYLineItem objects that only contain the variant ID (if created from a BUYProductVariant).

  variant

The BUYProductVariant object associated with the line item when created using the preferred initWithVariant: initializer.
